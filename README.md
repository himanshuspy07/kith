
# kith

A modern, professional real-time chat application built with Next.js 15, Firebase, and Tailwind CSS.

## Features
- **Real-time Messaging**: Private and group conversations powered by Firestore.
- **Dynamic Theming**: Support for Light, Dark, and System modes.
- **Rich Interaction**: User mentions (@username), message replies, and emoji reactions.
- **Professional UX**: Glassmorphism UI, infinite scroll message history, and secure authentication.
- **Cross-Platform**: Mobile-first responsive design.

## Technical Stack
- **Framework**: Next.js 15 (App Router)
- **Database/Auth**: Firebase (Firestore & Authentication)
- **Styling**: Tailwind CSS & ShadCN UI
- **Icons**: Lucide React
